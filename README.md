# Blog Platform

A simple full-stack blog platform developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. Users can register, log in, create, edit, delete blog posts, and add comments.

## Features
- User Registration & Login
- Create, Edit & Delete Blogs
- Comment System
- REST APIs
- MongoDB Integration

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
